# Alisa Tishchenko

Персональный сайт-портфолио фронтенд-разработчика Алисы Тищенко.

## Описание сайта

Сайт представляет профессиональный профиль, ключевые навыки и примеры проектов. Приложение выполнено как статический Next.js-проект и ориентировано на быстрый просмотр на разных устройствах.

## Возможности

- Главная страница с кратким позиционированием и CTA-переходом к работам.
- Блок `About` с описанием опыта и технологических компетенций.
- Блок `Work` с карточками проектов, стеком и внешними ссылками на репозитории.
- Блок `Contact` с быстрыми способами связи (email и GitHub).
- Навигация по якорным секциям (`About`, `Work`, `Contact`).
